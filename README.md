# Belle & Carrie - Rehabilitation Yoga Website

A serene, multi-page frontend website designed for a Yoga studio. This project focuses on clean typography, specialized color palettes, and a structured layout using **CSS Grid**.

## 🧘 Project Overview

This website provides a professional online presence for a yoga rehabilitation center. It features dedicated sections for class descriptions, instructor bios, and a community blog.

## 🚀 Key Features

- **Multi-Page Navigation:** Separate layouts for Home, About, Classes, Contact, and Blog.
- **Custom Typography:** Integrated Google Fonts (**Nixie One**) for a minimalist, airy feel.
- **CSS Grid System:** - Main content uses a `2fr 1fr` split for text and imagery.
  - The navigation menu and footer use grid-template columns for perfect alignment.
- **Social Media Integration:** Branded footer with links to Facebook, Google, Pinterest, and Twitter.
- **Responsive-Ready:** Built with a grid foundation that adapts easily to different screen sizes.

## 🛠️ Technologies Used

- **HTML5:** Semantic structure with clear class naming conventions.
- **CSS3:** - CSS Grid (Primary layout engine)
  - Flexbox (Social icon alignment)
  - Google Fonts API
